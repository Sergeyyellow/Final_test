Git

  • git status - Проверка статуса файла.
  • git push - Внесение изменения на удаленный репозиторий.
  • git commit -m "название коммита" - Создание коммита.
  • git log - Отоброжение списка всех коммитов.
  • git clone ссылка с GitHub - Клонирование удаленного репозитория на компьютер/
